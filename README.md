## Combo

To create a Combo, run the ComboEdit file, with this tool you can edit your Combo and add Host and Port to it. 

Your combo: EMAIL:PASS

Your Combo after using ComboEdit : HOST|PORT|EMAIL|PASS

```
python ComboEdit.py
input : Address Combo
input2 : Your Host
input3 : Your Port
```
